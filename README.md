# Student Management System

## 📌 Project Overview
The Student Management System is a web-based CRUD application developed using Spring Boot, Thymeleaf, and MySQL. The system allows users to manage student records including adding, updating, viewing, and deleting student information.

## 🚀 Features
- Add Student
- View Student List
- Update Student Details
- Delete Student Record
- Form Validation
- Database Integration with MySQL

## 🛠️ Technologies Used

Backend
- Java
- Spring Boot
- Spring MVC
- Hibernate / JPA

Frontend
- HTML
- CSS
- Bootstrap
- Thymeleaf

Database
- MySQL

Tools
- IntelliJ / Eclipse

## 🏗️ Architecture

MVC Architecture

Controller → Handles Requests  
Service → Business Logic  
Repository → Database Interaction  
View → Thymeleaf Templates

## ▶️ How to Run the Project

1. Clone the repository
2. Open project in IDE
3. Configure MySQL database
4. Update `application.properties`
5. Run Spring Boot Application
